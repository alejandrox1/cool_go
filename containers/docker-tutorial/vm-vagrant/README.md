# Vagrant

* For a default `Vagrant` file,
  ```
  vagrant init
  ```

* Turn on your machine
  ```
  vagrant up
  ```

* Default username and password: `vagrant:vagrant`.
  ```
  vagrant ssh
  ```

* To suspend the machine
  ```
  vagrant suspend
  ```

* To turn it off,
  ```
  vagrant halt
  ```

* To destroy,
  ```
  vagrant destroy
  ```
